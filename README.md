# 🌿 Panambas - E-Commerce Aplikasi (UAS)

Aplikasi E-Commerce berbasis Web yang dibangun menggunakan **Laravel** untuk memenuhi tugas Ujian Akhir Semester (UAS). Aplikasi ini menyediakan platform untuk transaksi jual-beli berbagai kebutuhan tanaman seperti bibit, tanaman hias, pupuk, dan media tanam.

## 🚀 Live Demo
Aplikasi ini sudah di-deploy dan dapat diakses secara publik melalui tautan berikut:
**[Kunjungi Web Panambas](https://uas-frontend-panambas-production.up.railway.app)**

## 🛠️ Tech Stack
* **Framework:** Laravel (PHP 8.2)
* **Database:** MySQL (Hosted on Railway)
* **Frontend:** HTML, CSS, JavaScript, Bootstrap/Tailwind
* **Deployment & Hosting:** Railway / GitHub

## ✨ Fitur Utama
* **Halaman Pengunjung & Pembeli:** Katalog produk, keranjang belanja (cart), proses checkout, riwayat pesanan, dan fitur *chat*.
* **Halaman Admin & Penjual:** Dashboard manajemen produk, manajemen kategori/sub-kategori, kelola pesanan (terima/tolak/kirim resi), dan manajemen pengguna.
* **Sistem Keamanan:** Autentikasi user (Login/Register) untuk pembeli dan admin.
